<h2>Sobre o CoelhoVPS</h2>
<p>O <strong>CoelhoVPS</strong> é um sistema exclusivo de gerenciamento de VPS, desenvolvido e mantido por <strong>Eronaldo Coelho</strong>. Este painel permite controlar servidores virtuais de forma completa, oferecendo recursos avançados para administração e automação.</p>

<h3>Funcionalidades Principais</h3>
<ul>
  <li>Painel administrativo moderno e seguro</li>
  <li>Gerenciamento de VPS: iniciar, parar, reiniciar e reinstalar servidores</li>
  <li>Snapshots e backups rápidos</li>
  <li>Acesso ao terminal dos servidores diretamente pelo painel</li>
  <li>Integração com MercadoPago para automação de pagamentos</li>
  <li>Logs detalhados de todas as ações no sistema</li>
</ul>

<h3>Tecnologias Utilizadas</h3>
<ul>
  <li><strong>Back-end:</strong> PHP (Laravel)</li>
  <li><strong>Front-end:</strong> Tailwind CSS, Vite</li>
  <li><strong>Banco de Dados:</strong> MySQL / MariaDB</li>
  <li><strong>Automação:</strong> Bash e Python</li>
</ul>
